**It's definitely unsecured but a quick way to do some magic.**

```php
<?php
#echo password to sudo
system('echo "PASS" | sudo -u root -S COMMAND');

#example
system('echo "password" | sudo -u root -S systemctl restart httpd.service')
?>
```
